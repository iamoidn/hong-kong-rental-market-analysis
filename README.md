# Hong Kong Rental Market Analysis

## Overview
This is a data analysis project that I built to examine 40+ years of Hong Kong's rental market trends using official government data (https://www.rvd.gov.hk/en/publications/property_market_statistics.html)It provides interesting insights into how the market trends have changed through the years, and also highlights major historical events that have shaped one of the world's most dynamic and expensive real estate markets.

All visualizations can be found in project_outputs directory

## Installation
1. Clone the repository:
```bash
git clone https://github.com/iamoidn/hong-kong-rental-market-analysis.git
cd hong-kong-rental-market-analysis
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

## How to Use:
```bash
python main.py
python clean_and_analyze.py
```

After running the code, all generated files will be placed in project_outputs directory.
